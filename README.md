## 🧠 What is Haystack?

Haystack is a custom Dependency Injection system designed for Unity that allows you to:

- Inject dependencies using attributes  
- Manage systems using keys/tags  
- Support both direct (static) and indirect (dynamic) injection  
- Handle runtime dependency changes cleanly  

---

## ✨ Key Features

- 🔹 Simple attribute-based workflow  
- 🔹 Direct & indirect injection support  
- 🔹 Runtime dynamic dependency handling  
- 🔹 Collider-based injection system  
- 🔹 Optional unbinding of dependencies
